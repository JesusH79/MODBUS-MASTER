# MODBUS_MASTER
## DESCRIOCION
ESTA APLICACION O PROGRAMA ESTA ECHO PARA PODER HACAER PRUEBAS CON LOS DISPOSITIVOS MODBUS,
YA SEA PARA: 
* LEER ENTRADAS (DISCRETE INPUTS-FC02) Y REGISTROS (INPUT REGISTERS-FC04);
* ESCRIBIR EN UN SOOLO REGISTRO(HOLDING REGISTER-FC06) O BOBINA(COILS-FC05);
* ESCRIBIR EN VARIOS REGISTROS(HOLDING REGISTERS-FC16) O BOBINAS(COILS-FC15);

ESTA APLICACION FUE HECHA EN C#, EN VISUAL STUDIO 2026 Y SE UTILIZO LA LIBRERIA EasyModbusTCP.
